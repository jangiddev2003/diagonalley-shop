🧙 Diagonalley OTP Tester

Test the forgot password API directly

1️⃣ Check Backend Connection

2️⃣ View All Users

3️⃣ Send OTP

4️⃣ Verify OTP

5️⃣ Reset Password

📝 Instructions

  1. Click "Test Backend Health" to verify connection
  2. Click "Get All Users" to see registered users
  3. Find your email in the list (or register first)
  4. Click "Send OTP" with your email
  5. Copy the OTP from the response
  6. Click "Verify OTP" and paste the code
  7. After verification, click "Reset Password"
  8. Try logging in with new password
